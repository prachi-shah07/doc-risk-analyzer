# 📄 AI Document Risk Analyzer

## 🚀 Overview
AI Document Risk Analyzer is a Python-based web application that extracts information from PDF documents and detects potential risks using rule-based NLP techniques.

This project simulates real-world document intelligence systems used in financial risk analysis.

---

## 🧠 Features

- 📄 Upload PDF documents
- 🔍 Extract text from documents
- 📊 Identify key fields (Amount, Date)
- ⚠️ Risk Detection (Low / Medium / High)
- 🧠 Summary generation
- 🌐 Interactive UI using Streamlit

---

## 🛠️ Tech Stack

- Python
- Streamlit
- pdfplumber
- Regex (NLP basics)
